 🌱 KrishiMitra – Frontend

KrishiMitra is a smart agricultural platform built to bridge the gap between farmers and dealers while providing easy access to resources, crop information, and support through an integrated FAQ chatbot.  

This repository contains the React.js frontend of the project.

---

 🚀 Features

- 👨‍🌾 Farmer Section  
  Farmers can register/login, Get the correct price prediction for their crops based on their location and market history and connect with dealers.  

- 🛒 Dealer Section 
  Dealers can log in, list their buying districts and connect with farmers directly.  

- 🤖 FAQ Chatbot  
  An integrated chatbot that answers frequently asked questions about farming, crop practices, and the platform itself.  

- 🔒 Authentication System  
  Secure login/signup for both Farmers and Dealers.  

- 🎨 Modern UI/UX
  Built with React, TailwindCSS, and Lucide Icons for a clean and responsive interface.  

---

🛠️ Tech Stack

- Frontend: React.js, React Router, Axios  
- UI: TailwindCSS, Lucide Icons, ShadCN components  
- State Management: React Context API  
- Backend (separate repo): Flask + MongoDB

cd krishimitra-frontend
2️⃣ Install dependencies
npm install
3️⃣ Start development server
npm start
The app will run on http://localhost:3000
